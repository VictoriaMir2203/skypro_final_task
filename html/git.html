Команды Git

  • git init - инициализация нового репозитория .
  • git clone - клонирование удалённого репозитория.
  • git status - показать текущее состояние файлов в репозитории.
  • git add - добавить файл в индекс для следующего коммита.
  • git commit -m "комментарий" - зафиксировать изменения с комментарием.
  • git push - отправить коммиты в удалённый репозиторий.
  • git pull - получить последние изменения из удалённого репозитория.
